*Projeto no Figma:* https://www.figma.com/design/lZS4CJRHWSztAuVpT6pAxS/TeslaBank?m=auto&t=ntficow91GtqrwWM-6

<br>

## 🌸 Desafios

<br>

Neste projeto, o foco foi dominar a organização de elementos em tela e a criação de componentes interativos:

- **CSS Grid Layout:** Utilizado na parte principal do site para dividir o conteúdo em duas colunas, alinhando textos e a imagem do cartão.
- **Flexbox:** Aplicado no cabeçalho (header) e nos botões para garantir que logos, links e ícones fiquem alinhados corretamente.
- **Variáveis CSS (:root):** Centralização de cores, tamanhos de fonte e escalas de espaçamento para facilitar a manutenção do código.
- **Efeitos Visuais:** Uso de drop-shadow personalizado para dar profundidade aos elementos, seguindo o guia de estilos do projeto.
